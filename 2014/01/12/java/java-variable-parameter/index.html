3、注意事项

3.1 在调用方法的时候，如果能够和固定参数的方法匹配，也能够与可变长参数的方法匹配，则选择固定参数的方法。

3.2 如果要调用的方法可以和两个可变参数匹配，则出现错误

3.3 一个方法只能有一个可变长参数，并且这个可变长参数必须是该方法的最后一个参数.

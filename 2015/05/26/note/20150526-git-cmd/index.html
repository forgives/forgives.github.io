clone指定分支的固定深度

–branch
让新克隆仓库的 HEAD 指向指定 的分支而不是默认仓库的 HEAD 。可以简化为 -b

–single-branch
只克隆一个单独的分支，由 --branch 指定分支

–depth [num]
depth用于指定克隆深度，为1即表示只克隆最近一次commit.

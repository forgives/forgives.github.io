mac zip命令

发表于 2018-03-15 更新于 2021-07-19 分类于 mac

简单用法

zip命令

参数 描述
-q 表示不显示压缩进度状态
-r 表示子目录子文件全部压缩为zip
-e 表示你的压缩文件需要加密，终端会提示你输入密码的
-m 表示压缩完删除原文件
-o 表示设置所有被压缩文件的最后修改时间为当前压缩时间

unzip命令

语法：unzip [选项] 压缩文件名.zip

各选项的含义分别为：

参数 描述
-x 文件列表 解压缩文件，但不包括指定的file文件
-v 查看压缩文件目录，但不解压
-t 测试文件有无损坏，但不解压
-d 目录 把压缩文件解到指定目录下
-z 只显示压缩文件的注解
-n 不覆盖已经存在的文件
-o 覆盖已存在的文件且不要求用户确认

1、将压缩文件text.zip在当前目录下解压缩

2、将压缩文件text.zip在指定目录/tmp下解压缩，如果已有相同的文件存在，要求unzip命令不覆盖原先的文件

3、查看压缩文件目录，但不解压

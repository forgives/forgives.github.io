ognl版本升级遇到的问题

ognl jar版本从3.1.19升级到3.2.13会遇到MemberAccess implementation must be provided!异常，原因是升级后作者取消了默认的memberAccess设置且强制必须传入memberAccess的实现。作者也自己写了一个 DefaultMemberAccess，可以直接使用（初始化的时候建议 allowAllAccess = true ）。 对应的版本更新说明 。

参见 https://blog.csdn.net/LX928525166/article/details/82699572

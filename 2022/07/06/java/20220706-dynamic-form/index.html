动态表单的技术方案。

（2022年12月13日更新: 最近看了AWT，用这个思路就可以了，没有必要自己创建一套）

前端动态表单技术方案 Alibaba Formily

服务端不直接生成formily schema ， 前后端使用独立的协议由前端转化为 formily schema。

协议模型

Json表达形式

  • name 标签的name，表单提交用
  • type 标签的类型，可以扩展
    • radio
    • checkbox
    • text
    • textarea
    • upload
    • select
  • title 展示标题
  • valueType 值的类型，多选的时候使用array
    • string
    • bool
    • array
  • required 是否必填
  • maxLength 最大值
  • enums [] 选项枚举
    • label
    • value

WoWSimpleRegistration

WoWSimpleRegistration 是PHP项目
支持 AzerothCore, TrinityCore, AshamaneCore, CMangos.

需要php >= 7.0 且依赖模块 gmp, gd, soap, mbstring, pdo and pdo-mysql.

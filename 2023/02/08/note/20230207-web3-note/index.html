Web 3.0发展趋势学习笔记

结论

  1. Web 3.0 时代用户对内容的主权将提升，平台模式与去中心化模式长期并存。
  2. Web 3.0 技术路线、商业模式与运用场景有较大不确定性，但是对基础的 计算、存储、AI、区块链等 前沿技术需求是明确的。

web x.0 时期的划分

按人与互联网互动方式的变化来区分。

  • Web 1.0 内容在平台，用户只能被动接收
  • Web 2.0 用户不仅限于被动接收，也能主动创造，但所有权仍在平台
  • Web 3.0 内容所有权在创作者，用户拥有内容。

但目前看Web 3.0 完全的去中心化过于激进，缺乏理论基础和实践，去中心化与平台长期共存。

web 3.0 特征

1.去中心化
2.无权限化
3.无边界网络

web 3.0 挑战

1.基础设施以及技术支撑不够： 基于区块链技术的去中心化生态，需要高吞吐低延迟的区块链网络，算力要求巨大。
2.应用和生态不成熟：需要额外的步骤、软件与教学，对广泛应用造成障碍
3.治理监管挑战

国内现状

DeFi（去中心化金融）、NFT（非同质化代币）、GameFi（引入DeFi的区块链游戏）、DeSo（分布式社交）

web 3.0 涉及的主要技术

区块链技术、分布式存储、分布式计算、人工智能等。

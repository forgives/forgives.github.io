在沉浸式翻译使用

  1. 访问官网下载浏览器插件
  2. 点击设置 -> 开发者设置 -> 开启Beta测试特性
  3. 基本设置 -> 翻译服务选择DeepLX(Beta) -> API URL 填入 https://api.deeplx.org/translate

就可以直接使用了。

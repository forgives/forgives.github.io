warp-plus

发表于 2024-04-13 更新于 2024-08-23 分类于 网络

warp-plus is an open-source implementation of Cloudflare’s Warp, enhanced with Psiphon integration for circumventing censorship.

This project aims to provide a robust and cross-platform VPN solution that can use psiphon on top of warp and warp-in-warp for changing the user virtual nat location.

Usage

exp:
warp-plus.exe -b 0.0.0.0:8890 -e 优选ip:端口号 -k g8lG9B50-Ge346l2I-5FeYv182 --country 国家代码 --cfon

warp-plus.exe -b 0.0.0.0:7899 -e 162.159.195.201:854 -k C2m39Lq5-o6L4Xh09-v9br1W34 --country US --cfon

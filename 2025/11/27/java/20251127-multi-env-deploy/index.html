发表于 2025-11-27 分类于 Java

一份代码aliyun 和 aws 部署的中间件接入实践。

背景

国内使用 aliyun 中间件的应用现在需要部署到 aws ，所有很多中间件需要做平替。并且因为是一份代码，需要支持国内外境部署。

核心思路

使用 @ConditionalOnProperty 等条件控制对接接口实现的实例生成来控制最终调用的是哪一个实现。

frii.site API 域名管理

发表于 2026-03-28

frii.site 是一个免费域名服务，提供简单的 API 接口来管理域名解析记录。

API 基础信息

  • API 地址: https://api.frii.site
  • 认证方式: 通过 X-API-Token 请求头认证
  • 官方文档: https://guides.frii.site/guides/api.html

Token 格式为 $APIV2=xxxxx，需要先在 frii.site 注册获取。
